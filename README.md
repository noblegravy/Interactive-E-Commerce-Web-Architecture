A modern e-commerce platform for custom PC sales and configuration, built with React and TypeScript. The application allows users to explore curated PC builds, compare specifications, customize configurations, and submit inquiries through a responsive and user-friendly interface.

## Overview

GAPR PC Solutions is designed to provide users with a streamlined experience when browsing and configuring custom-built PCs. The platform showcases multiple product categories, highlights system specifications, and enables customers to submit build requests and inquiries.

The project emphasizes clean UI design, component-based architecture, and responsive web development principles.
